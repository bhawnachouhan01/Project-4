# Project-4
# 📱 Feature Extraction and Price Prediction for Mobile Phones

A Machine Learning project that predicts smartphone prices using technical specifications and feature engineering. The project includes data preprocessing, exploratory data analysis (EDA), feature importance analysis, and regression model comparison to identify the best-performing model.

---

## 📌 Project Overview

The objective of this project is to build a machine learning model that predicts smartphone prices based on their hardware and software specifications.

The project follows a complete data analytics workflow:
- Data Preprocessing
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Model Development
- Model Evaluation
- Feature Importance Analysis
- Business Recommendations

---

## 🎯 Objectives

- Predict smartphone prices using Machine Learning.
- Identify the most influential pricing features.
- Compare Linear Regression and Random Forest Regression.
- Generate business insights from model results.

---

## 📂 Dataset

- **Total Records:** 531 Smartphones
- **Target Variable:** Price (₹)
- **Input Features:** 14 Selected Features

Example Features:
- Brand
- Model
- RAM
- Memory
- Processor
- Battery Capacity
- Front Camera
- Rear Camera
- Total Camera MP

## 🤖 Machine Learning Models

- Linear Regression
- Random Forest Regression

---

## 📈 Model Performance

After comparing both models:

✅ **Random Forest Regression** achieved the best performance with:
- Lower MAE
- Lower MSE
- Lower RMSE
- Higher R² Score

Therefore, **Random Forest Regression** was selected as the final model.

---

## 💡 Key Insights

- Camera quality significantly influences smartphone prices.
- Brand and Memory are strong pricing factors.
- RAM and Battery Capacity also contribute to price prediction.
- Feature Engineering improved overall model performance.

---

## 🚀 Business Recommendations

- Improve camera quality for premium devices.
- Optimize RAM and battery capacity.
- Use Machine Learning for pricing decisions.
- Update the model regularly with new smartphone data.
